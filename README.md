
# Pyn Technologies Website

This is the official website for Pyn Technologies, showcasing our services in Website & App Development, Digital Marketing, AI Solutions, WhatsApp Business API, UI/UX Design, and Graphics Designing.

## Project Info

This project is built with modern web technologies to ensure optimal performance and user experience.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- UI Components
- Tailwind CSS

## Development

To run this project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Features

- Responsive design for all devices
- Modern UI/UX with smooth animations
- Optimized performance
- Comprehensive showcase of services and solutions

## Custom Domain

To connect a custom domain to this site, update your DNS settings to point to our hosting servers.
